# Cybertanks

## Demo

From the root project directory, run the following gradle tasks:

```
gradle build :arcade-main:jvmRun
```

Use the UI to load the cartridge at `cybertanks-cartridge/build/libs/cybertanks-cartridge-cartridge.jar` and the
controller at `cybertanks-sample/build/libs/cybertanks-sample-jvm.jar`.
